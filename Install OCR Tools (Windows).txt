Invoice Splitter — Install Poppler + Tesseract on Windows
=========================================================

You need BOTH for scanned PDFs. Text-only PDFs work without them.

--- TESSERACT (OCR) ---

1. Download installer:
   https://github.com/UB-Mannheim/tesseract/wiki

2. Run the installer (admin may be required).

3. On the install screen, note the folder (default):
   C:\Program Files\Tesseract-OCR

4. IMPORTANT: Check the box "Add to PATH" if the installer offers it.
   If there is no checkbox, add PATH manually (step 5).

5. Manual PATH (if needed):
   - Press Windows key, type "environment variables", open
     "Edit the system environment variables"
   - Environment Variables → under "User variables" select Path → Edit
   - New → paste:  C:\Program Files\Tesseract-OCR
   - OK on all dialogs
   - Close and reopen any Command Prompt / setup windows

6. Test in Command Prompt:
   tesseract --version

--- POPPLER (PDF → images) ---

WRONG download (no .exe, no Library folder):
  https://poppler.freedesktop.org/  — that is SOURCE CODE (e.g. poppler-26.05.0.tar.xz)

RIGHT download (pre-built Windows .exe files):
  https://github.com/oschwartz10612/poppler-windows/releases/
  File name looks like:  Release-26.02.0-0.zip   (under Assets, NOT "Source code")

1. Download that Release-....zip only.

2. Extract to e.g. C:\poppler

3. Open the extracted folder. You should see:
     poppler-26.02.0\Library\bin\pdftoppm.exe
   (version number in folder name may differ)

   Use Windows search inside the extracted folder for:  pdftoppm.exe
   The folder that contains pdftoppm.exe is what you add to PATH.

3. Add Poppler to PATH:
   - Same Environment Variables steps as above
   - New → paste:  C:\poppler\Library\bin
   - OK on all dialogs
   - Close and reopen Command Prompt

4. Test in Command Prompt:
   pdftoppm -h

--- FINISH ---

1. Double-click "Setup Invoice Splitter.bat" again.
   Poppler and Tesseract should show [ OK ].

2. If still red but files exist in the folders above:
   - Run "Fix Setup (Windows).bat" after git pull (auto-detects default folders)
   - Or log off Windows and back on (refreshes PATH)

--- NO ADMIN RIGHTS? ---

Ask IT to install Tesseract-OCR and Poppler, or install to your user folder
and add only your user PATH (no admin needed for user PATH).
